Web Dice Roller

Roll five 6-sided dice (Yahtzee-style). Fast, clean, no drama.

How to use

  1. The dice roll automatically when the page loads.
  2. Press Enter (or click Roll) to roll again.
  3. All dice values are read-only (because dice don’t take suggestions).

Tip: After rolling, the Roll button stays focused — so Enter keeps rolling.
